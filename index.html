Очистка лишних метаданных из фото и видео

Этот локальный HTML-сайт обрабатывает файлы прямо в браузере. Для фото он пересобирает изображение заново через Canvas, что обычно убирает EXIF, GPS, XMP, ICC, комментарии и прочие лишние поля. Для видео он использует FFmpeg WebAssembly и пересобирает контейнер, удаляя метаданные потоков и контейнера настолько глубоко, насколько это возможно в браузере.

Локально в браузере
Без отправки файлов на сервер
Фото: EXIF/GPS/XMP удаляются
Видео: контейнерные метаданные удаляются

Загрузка файлов

Можно перетащить сюда файлы или выбрать вручную. Поддерживаются изображения и видео.
Перетащите фото или видео сюда
или выберите файлы кнопкой ниже
Для фото сайт сохраняет только «пиксели» и базовый формат вывода. Для видео сайт пересобирает контейнер и удаляет метаданные контейнера/потоков. Но некоторые особенности исходного кодека, таймингов, цветовых параметров или встроенных служебных данных могут быть неотделимы без перекодирования. Поэтому для видео это максимально строгая очистка, доступная в браузере, но не абсолютная математическая гарантия удаления вообще любого возможного служебного бита во всех форматах.
Готово.\n

Что именно делает сайт

Фото: изображение декодируется и заново экспортируется. Так метаданные не копируются из исходного файла в новый.

Видео: FFmpeg в браузере запускается локально, файл пересобирается с ключами удаления метаданных. По умолчанию выполняется попытка без перекодирования там, где это возможно, чтобы сохранить качество и скорость.

Приватность: файл не уходит на сервер. Всё происходит на вашем устройстве.

Ограничения

1. Очень большие видео могут потребовать много памяти.

2. При открытии по file:// некоторые браузеры могут ограничивать загрузку модулей. Надёжнее запускать через простой локальный сервер.

3. FFmpeg WebAssembly подгружается из CDN. Если нужен полностью автономный офлайн-вариант, можно сохранить рядом файлы ядра FFmpeg и заменить пути в коде.

Самый простой вариант:

python -m http.server 8000

Потом откройте:
http://localhost:8000
Совет: после очистки проверьте итоговые файлы независимым просмотрщиком метаданных. Для фото результат обычно «чистый». Для видео степень очистки зависит от формата контейнера и особенностей исходного файла.
